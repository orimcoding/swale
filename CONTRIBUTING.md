# Contributing to Swale

Swale is a personal project I'm building and maintaining on my own. I'm making it public so anyone can follow along, learn from it, or use it — but I'm not currently looking for active contributors.

That said, I'm always open to feedback. If you spot a bug, have a suggestion, or want to share an idea, feel free to open an issue. I read everything.

---

## If you want to suggest something

Open an issue using the bug report or feature request template. Be specific about what you're seeing or what you'd like to see. I'll respond when I can.

## If you want to run it locally

Follow the setup steps in [README.md](README.md).
